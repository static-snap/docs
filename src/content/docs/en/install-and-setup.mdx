## Introduction

Static Snap is a powerful plugin that transforms your dynamic WordPress site into a static website. This guide will walk you through the installation and setup process so you can quickly get started using Static Snap.

### Download Static Snap

You can download Static Snap directly from our website or [install it through your WordPress admin panel](#direct-installation-via-wordpress-admin):

- [Download Static Snap](https://staticsnap.com/download)

We recommend using the WordPress admin panel for a seamless and straightforward installation process.

:::tip[Learn More About WordPress Plugins]
If you're new to managing plugins in WordPress, you can visit the official [Manage Plugins guide](https://wordpress.org/documentation/article/manage-plugins/) for more information.
:::

## Prerequisites

Before you begin, ensure that your environment meets the following requirements:

- **PHP**: Version `7.4` or higher.
- **WordPress**: Version `6.5` or higher.

## Installation Methods

There are two ways to install Static Snap on your WordPress site:

### Direct Installation via WordPress Admin
You can install it directly through your WordPress admin panel. Follow these steps:

<Steps>
1. **Log in to Your Admin Dashboard**: Access your WordPress admin panel. Navigate to Plugins Section: Go to Plugins > Add New.
2. **Search for Static Snap**: In the search bar, type `staticsnap` and press enter.
3. **Install the Plugin**: Locate Static Snap in the results and click the Install Now button.
4. **Activate the Plugin**: Once the installation is complete, click Activate to enable Static Snap on your site.
</Steps>

### Installing Static Snap Manually

Follow these steps to manually install the Static Snap plugin on your WordPress site:

<Steps>
1. **Download the Plugin**: Download the latest version of Static Snap from [our website](https://staticsnap.com/download).

2. **Access WordPress Admin Panel**: Log in to your WordPress admin dashboard.

3. **Navigate to Plugins Section**: Go to `Plugins` > `Add New`.

4. **Upload the Plugin**: Click on the `Upload Plugin` button at the top of the page.

5. **Select the Plugin File**: Choose the Static Snap plugin file you downloaded, and click `Install Now`.

6. **Activate the Plugin**: Once the installation is complete, click on `Activate Plugin` to enable Static Snap on your WordPress site.
</Steps>
